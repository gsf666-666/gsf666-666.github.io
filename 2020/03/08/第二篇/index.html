关于如何利用未root的旧手机搭建个人博客

三月 08, 2020

请在手机上先下载一个termux。点此(https://termux.com/)
话不多说直接上步骤:
1.更新一下 pkg install upgrade 然后 apt install upgrade
2.安装必要工具 pkg install git vim nodejs
3.安装hexo npm install -g hexo-cli
4.运行 hexo init （随便什么）
安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init
$ cd
$ npm install
新建完成后，指定文件夹的目录如下：

.
├── _config.yml
├── package.json
├── scaffolds
├── source
| ├── _drafts
| └── _posts
└── themes
5.现在你可以运行 hexo clean 来清理一下缓存
6.运行 hexo g 来生成一下网页
7.运行hexo s
现在你可以通过浏览器来实现本地访问了

下面的步骤是为了实现公网访问
1.下载cpolar curl -O -L https://www.cpolar.com/static/downloads/cpolar-stable-linux-arm.zip
2.解压压缩包 unzip cpolar-stable-linux-arm.zip
3.登录官网（https://www.cpolar.com）注册并登录复制你的token值
4.将你刚刚复制的粘贴到termux并且回车
现在你已经可以开始你的表演了😄
关闭termux再打开termux
在你当前的见面上运行hexo s然后打开另一个窗口（从左边划出一个框框，长按右下角的来新建一个窗口）在这个窗口运行 ./cpolar http 4000
好了现在你已经可以在任何一个地方打开你的博客
